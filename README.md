Stress Prediction System using K-Nearest Neighbors (KNN)
Project Overview

The Stress Prediction System is a machine learning application that predicts an individual's stress level based on various personal, academic, and lifestyle factors. The prediction model is built using the K-Nearest Neighbors (KNN) algorithm, while the frontend is developed using Streamlit, providing a simple and interactive web interface for users.

The application enables users to enter relevant information through an easy-to-use interface and instantly receive a stress level prediction.

Features
Predicts stress levels using the K-Nearest Neighbors (KNN) algorithm.
Interactive and user-friendly web interface built with Streamlit.
Real-time prediction based on user inputs.
Simple and responsive design.
Fast and lightweight application.
